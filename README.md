# 🚀 AI Resume Ranker

An AI-powered Resume Ranking System that analyzes and ranks resumes based on a given job description using NLP techniques like TF-IDF Vectorization and Cosine Similarity.

Designed with a modern dark-themed UI using Streamlit.

---

# 📌 Features

- 📄 Upload Multiple Resumes (PDF, DOCX, TXT)
- 🧠 AI/NLP-based Resume Ranking
- 📊 Score-based Candidate Comparison
- 🎯 Strengths & Gap Analysis
- 🌙 Modern Dark UI
- ⚡ Fast and Efficient Processing
- 🔒 Privacy-Focused Design
